# Advanced controls

Library of advanced Blade components for your Laravel website.
Design gorgeous websites with minimal effort.


## Installation

Run the following command from your project's root directory:
```
composer require allyoullneed/advanced-controls
```

## How to use

Add the components in your template like you would do with any other.

Get the list of available components and documentation from our [website](https://allyoullneed.com).
